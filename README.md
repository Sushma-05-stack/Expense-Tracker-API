# MERN CRUD App

This is a full-stack CRUD application built with MongoDB, Express.js, React, and Node.js.

## Prerequisites

- Node.js installed
- A MongoDB cluster
- A third-party SDK API Key (e.g., OpenAI, Stripe)

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Create a `.env` file based on `.env.example` and add your credentials.

3. Run the application (starts both frontend and backend):
   ```bash
   npm run dev
   ```

## Architecture
- **Frontend**: React (Vite) running on port 3000
- **Backend**: Express running on port 5000
